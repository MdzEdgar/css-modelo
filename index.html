Ejercicio #1 Caja Estandar

Instrucciones

En este ejercicio vamos a practicar el modelo de caja estandar.

  • 1.- Crear una caja padre con las siguientes características:
    • 500px de alto
    • 500px de ancho
    • Borde discontinuo (dashed)
  • 2.Crear 4 cajas hijos con las siguientes características:
    • Cada una de las 4 cajas tendrá un padding de 10px
  • 3. Dar características a las cajas hijo de la siguiente manera
    • Caja 1 debe de tener una imagen de fondo(background-image)
    • Caja 2 debe de tener un fondo 'red'
    • Caja 3 debe de tener un color de fondo 'salmon'
    • Caja 4 debe de tener un color de fondo 'green'
    • Caja 1 y caja 2:
      • Borde superior e inferior de 10px de color 'grey'
    • Caja 3 y caja 4:
      • Borde derecho e izquierdo de 10px color 'cyan'
  • 4. El ejercicio debe de cumplir con los siguientes requisitos
    • No se puede utilizar la propiedad overflow
    • No se puede utilizar la propiedad overflow
    • Las 4 cajas hijo deben de tener las mismas medidas de alto y deancho(sumando bordes y paddings) ocupando cada caja 1/4 delespacio del contenedor padreo
    • Todas las medidas tienen que ser asignadas en pixeles
